## Objetivo

En el presente laboratorio tiene como objetivo aplicar herramientas planeación usando el frameworks agile Scrum con la tecnología JIRA.
A partir de un caso de estudio práctico se les permitirá a los estudiantes apropiar la tercera fase del ciclo de vida del desarrollo de software.
El laboratorio se desarrollará en los mismos equipos del Laboratorio #3.
